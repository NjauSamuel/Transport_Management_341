CTP Briefs

System Reports

Department-wise Trip Summary

This report provides a summary of trips for each department, Including the department name, total number of trips, total budget allocated, and average budget per trip.

Upcoming Trips Overview

This report displays a list of upcoming trips along with their details such as visit location, date, and purpose. It includes information about the department and the approval status of the trip.

Transportation Utilization Report

Shows the utilization of different types of transport vehicles, including information about the number of trips each vehicle has been assigned to and the total capacity utilized.

Funding Requests Summary

This report provides a summary of funding requests made by departments, including the total number of requests, total amount requested, and average amount per request.

Communication Log

Displays a log of communications between departments regarding trip planning, including details such as the communication date, sender department, receiver department, and the content of the communication.

Approval Workflow Overview

Shows an overview of the approval workflow for trips, including information about which trips have been approved by the DVC AA and the current status of each trip in the approval process.
